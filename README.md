# smartnet-monitor

Ce projet permet de :

Analyser le trafic réseau en capturant les paquets (qui communique avec qui, sur quels ports, etc.)
Détecter des comportements suspects (intrusions, trafic anormal, problèmes réseau) grâce à un modèle de Machine Learning
Afficher les résultats dans un dashboard web simple pour visualiser les anomalies et les statistiques

Le projet consiste à créer un outil de surveillance réseau intelligent, capable de détecter automatiquement des anomalies grâce à l’intelligence artificielle.
L’objectif principal est de proposer une solution accessible 100% gratuite pour surveiller un réseau informatique surtout pour les petites structures qui n’ont pas les moyens d’utiliser des outils professionnels coûteux.
